Important FAQ

What is Document Object Model (DOM) ?

(DOM) is a programming interface that allows developers to interact with and manipulate HTML and XML documents

How do you select and element from the DOM?

DOM (Document Object Model) by its ID, class, tag name, attributes, or a combination of these.

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received
